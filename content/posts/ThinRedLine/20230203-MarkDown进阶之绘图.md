> markdown基础用法: 参考

## 一、整体结构

```txt
[图形类型及相关配置]
    [图形内容]
```

|选项|描述|
|---|---|
|类型及相关配置|声明图形类型及其声明，例如`mermaid TB;`|
|图形内容|相关的图形内容文本|

> 引入流程图的方式各有不同，原生使用` ```mermaid`方式引入，而本站使用基于LoveIt主题的hugo引入方式为模板

## 二、流程图

### 2.1 基础

```txt
graph [流程图方向]
    [流程图内容]
```

```txt
graph LR
    A --> B
```

{{< mermaid >}}
graph LR
    A --> B
{{</mermaid>}}

### 2.2 流程图方向[^1]

|标志|含义|解释|
|---|---|---|
|TB|top bottom|从上到下|
|BT|bottom top|从下到上|
|RL|right left|从右到左|
|LR|left right|从左到右|
|TD||等同于TB|
